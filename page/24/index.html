互联网主流领域

大数据、云计算、人工智能、物联网

AI 主流领域

自然语言处理（NLP）、计算机视觉（CV）、推荐系统 / 计算广告

计算机视觉

  • OpenGL（Open Graphics Library）- 开放图形库
  • OpenCL（Open Computing Language）- 开放计算语言
  • OpenCV（Open Source Computer Vision Library）- 开放计算机视觉库
  • Detectron - 物体检测研究平台

人工智能

  • OpenAI
  • SINGA-Apache 开源的深度学习框架
  • MXNet-Apache 开源的深度学习框架
  • Tensorflow - 谷歌开源的机器学习框架
  • PaddlePaddle - 百度开源的深度学习平台
  • Keras - 基于 Python 的深度学习框架
  • PyTorch（Tensors and Dynamic neural networks in Python with strong GPU acceleration）
  • Horovod（Distributed training framework for TensorFlow, Keras, PyTorch, and Apache MXNet）

前端

  • Hexo - 静态博客框架
  • APlayer-HTML5 音乐播放器
  • DPlayer-HTML5 视频播放器

分布式数据库

  • Apache Ignite - 以内存为中心的分布式数据库，缓存和处理平台
  • ShardingSphere - 开源的分布式数据库中间件解决方案组成的生态圈，它由 Sharding-JDBC、Sharding-Proxy、Sharding-Sidecar 组成

RPC 框架

  • Tars - 腾讯开源的 RPC 框架
  • BRPC - 百度开源的 RPC 框架

Web 服务器

  • Openresty - 将 Nginx 转变为完整的可编写脚本的 Web 平台

Java 工具库

  • Arthas - 阿里巴巴的 Java 诊断工具
  • VJTools - 唯品会内部的 Java 编程规范、库、工具

数据库

  • SQLAdvisor - 美团开源的 SQL 优化工具

应用监控

  • CAT - 美团基于 Java 开发的实时应用监控平台

压测工具

  • TCPCopy-A TCP Stream Replay Tool

爬虫

  • awesome-java-crawler-Java 爬虫资源整理
  • Selenium - 一个用于 Web 应用程序测试的工具
  • Crawlab - 分布式爬虫管理平台，支持任何语言和框架
  • Puppeteer - 谷歌开源的 Node 库，可以用来模拟 Chrome 或者 Chromium 浏览器的运行

认证、权限

  • JustAuth - 史上最全的整合第三方登录的开源库

缓存

  • Overlord - 哔哩哔哩基于 Go 语言编写的 memcache 和 redis&cluster 的代理及集群管理系统

MyBatis 代码生成

  • mybatis
  • mybatis generator
  • pageHelper
  • mybatis-generator-gui
  • codehelper.generator
  • mybatis generator plugin
  • mybatis plus generator

Chrome 插件

  • Chrome 插件英雄榜

Markdown 静态博客编辑器

  • Gridea
