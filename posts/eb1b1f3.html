Maven 代码格式化插件

Spring Java Format 插件集提供了一款格式化 Java 代码的 Maven 插件，默认使用 Spring 的代码风格，插件的运行依赖于 JDK 11+。在项目里配置好代码格式化的 Maven 插件后，可直接运行命令格式化项目代码： mvn spring-javaformat:apply 或者 ./mvnw spring-javaformat:apply。当然也可以在 IDEA 中安装 Spring-Javaformat 插件进行自动格式化，详细的使用教程请看 官方文档。

基础使用

现在可以使用 mvn spring-javaformat:apply 或者 ./mvnw spring-javaformat:apply 命令格式化 Java 代码了。

插件安装

  • 1、打开 IDEA 插件市场的界面
  • 2、搜索 CheckStyle-IDEA，点击安装即可

插件配置

配置插件

  • 1、打开 CheckStyle 的配置界面（File -> Settings -> Tools -> Checkstyle）
  • 2、选择 Checkstyle 版本为 8.39
  • 3、在界面上点击配置文件的添加按钮，配置描述可随便填写（例如 Custom Checks），然后选中项目里的 checkstyle.xml 配置文件，点击下一步和完成
  • 4、在界面上勾选刚刚添加的配置文件

配置编辑器

  • 1、打开 IDEA 编辑器的配置界面（File -> Settings -> Editor -> Code Style）

插件使用

  • 1、在 IDEA 界面内打开任意一个 Java 源文件
  • 2、在 IDEA 界面的底部点击打开 CheckStyle 的操作面板，点击左侧的 绿色三角形 按钮，这样就可以检查代码风格

值得一提的是，CheckStyle 除了可以检测单个 Java 源文件的代码风格，还支持检测整个模块或者整个项目的 Java 代码风格
