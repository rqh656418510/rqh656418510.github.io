前言

本文主要介绍 Spring + Spring MVC + MyBatis 整合（SSM），开发工具基于 Eclipse + Maven。值得一提的是，下文只给出 SSM 整合所需的最小配置内容，在生产环境需要适当优化项目的配置，尤其是 Log4j2、Druid 的配置。完整的项目代码可以直接从 GitHub 下载对应章节 ssm-study。
