Docker 启动容器每次都拉取镜像

执行 docker run 命令启动容器时，不会自动拉取最新的镜像。它会直接使用本地已有的 latest 版本镜像，而不会从远程 Docker Hub 仓库拉取最新的镜像。为了每次启动容器都自动拉取最新的镜像，可以参考以下命令：

方式是否拉取最新镜像
docker run -d ...❌ 不会
docker pull ... && docker run -d ...✅ 会拉取
docker run --pull=always -d ...✅ 会拉取
docker-compose 配置 pull_policy: always✅ 会拉取
