Git 撤销指定的提交

使用 git revert 命令可以撤销（反做）某个已提交的 commit，但不修改历史记录，而是创建一个新的 commit 来 “抵消” 原来的变更。

Git 恢复到某个提交的历史版本

此操作适用于文件的删除或修改操作 commit 之后，想找回已删除的文件，或者将文件还原到修改前的某个历史版本，成功的前提是被删除或者被修改的文件必须曾经 commit 到本地库。
