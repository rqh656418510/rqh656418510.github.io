构建环境

  • npm 6.5.0
  • node 10.15.0
  • webpack 1.13.2
  • debian 9 (stretch)

目录结构

  • layout - 模板目录
  • languages - 语言配置目录
  • source-src - 源文件目录，编译到 source 目录
  • source - Hexo 加载主题资源的主目录，需要编译生成

一般来说，如果想修改页面的 html，可以到 layout 文件夹里直接修改。如果想修改 css、js，则需要到 source-src 文件夹里修改，并通过后面介绍的构建步骤，将源码编译到 source 目录下。
